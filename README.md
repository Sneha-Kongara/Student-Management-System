# Student-Management-System


## Overview

Student Management System is a desktop application developed using Java Swing, JDBC, and MySQL. It helps manage student records through a graphical user interface.

## Features

* Admin Login
* Add Student
* Update Student
* Delete Student
* Search Student
* View Student Records

## Technologies Used

* Java
* Swing
* JDBC
* MySQL
* Eclipse IDE

## Database Setup

1. Open MySQL Workbench.
2. Create a database named `student_management`.
3. Run the SQL script located in the `sql` folder.

## How to Run

1. Import project into Eclipse.
2. Add MySQL Connector JAR to Build Path.
3. Configure database username and password in `DBConnection.java`.
4. Run `Main.java`.
